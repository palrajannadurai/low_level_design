# Low Level Design - Design Patterns

This repository contains simple Java implementations of commonly used **Design Patterns** for learning **Low Level Design (LLD)**.

## Implemented Patterns

### Creational
- **Singleton** – Ensures only one instance of a class exists.
- **Factory** – Creates objects without exposing the creation logic.

### Structural
- **Adapter** – Allows incompatible interfaces to work together.
- **Decorator** – Adds new behavior to objects dynamically.

### Behavioral
- **Strategy** – Defines interchangeable algorithms and selects them at runtime.
- **Observer** – Notifies multiple objects when a state changes.
- **Command** – Encapsulates a request as an object.

## Additional Problem
- **Stock Price Alert System** – Example system using design principles.

## Purpose
Practice **Low Level Design concepts and design pattern implementations in Java**.
